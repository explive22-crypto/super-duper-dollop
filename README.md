# super-duper-dollop

Учебный проект простого приложения-трекера для начинающего разработчика.

Идея проекта: сделать небольшое приложение, в котором можно добавлять задачи или привычки, отмечать прогресс и видеть, что уже выполнено.

## Что будет в приложении

- список задач или привычек;
- добавление новой записи;
- отметка записи как выполненной;
- простая статистика прогресса;
- понятная структура кода, которую легко расширять.

## Для кого проект

Проект рассчитан на новичка. Здесь не будет сложной архитектуры на старте: сначала важнее понять, как устроены файлы приложения, где хранится логика и как постепенно добавлять новые возможности.

## Возможный первый шаг

Начать можно с простой HTML-страницы или небольшого JavaScript-приложения:

1. показать список задач;
2. добавить форму для новой задачи;
3. научиться отмечать задачу выполненной.

После этого проект можно будет развивать: добавить сохранение данных, фильтры, счетчики и оформление.
