# Идея приложения-трекера

## Краткое описание

`super-duper-dollop` — это простой учебный трекер задач или привычек.

Пользователь сможет записывать небольшие дела, отмечать выполненное и наблюдать за своим прогрессом. Проект подходит для тренировки базовых навыков: работа со структурой проекта, интерфейсом, состоянием данных и простыми пользовательскими действиями.

## Основные сценарии

- пользователь открывает приложение и видит список записей;
- пользователь добавляет новую задачу или привычку;
- пользователь отмечает запись как выполненную;
- приложение показывает, сколько записей выполнено.

## Минимальная версия

В первой версии достаточно сделать:

- поле ввода;
- кнопку добавления;
- список записей;
- кнопку или чекбокс для отметки выполнения.

## Что можно добавить позже

- удаление записей;
- фильтры: все, активные, выполненные;
- сохранение данных в браузере;
- даты и категории;
- небольшую статистику за день или неделю.

## Учебная цель

Главная цель проекта — не собрать сложный продукт, а пройти понятный путь от пустой папки до работающего маленького приложения.
