# Как запустить и проверить приложение

Это простое браузерное приложение. Для него не нужно устанавливать зависимости или запускать сервер.

## Как открыть приложение

1. Открой папку проекта.
2. Перейди в папку `app`.
3. Открой файл `index.html` двойным кликом или через браузер.

После этого на странице появится трекер задач.

## Как проверить добавление задачи

1. Введи текст задачи в поле ввода.
2. Нажми кнопку `Добавить`.
3. Проверь, что задача появилась в списке.

## Как проверить отметку выполненной задачи

1. Добавь задачу, если список пустой.
2. Нажми на чекбокс рядом с задачей.
3. Проверь, что текст задачи стал зачёркнутым.
4. Проверь, что счётчик `Выполнено` изменился.

## Как проверить фильтры

1. Добавь несколько задач.
2. Отметь одну задачу выполненной.
3. Нажми `Все`, чтобы увидеть все задачи.
4. Нажми `Активные`, чтобы увидеть только невыполненные задачи.
5. Нажми `Выполненные`, чтобы увидеть только выполненные задачи.

Если в выбранном фильтре задач нет, приложение покажет текст `Задач пока нет`.

## Как проверить удаление задачи

1. Добавь задачу или выбери уже существующую.
2. Нажми кнопку `Удалить` рядом с задачей.
3. Проверь, что задача исчезла из списка.

## Как проверить сохранение после перезагрузки

1. Добавь одну или несколько задач.
2. Отметь часть задач выполненными.
3. Перезагрузи страницу в браузере.
4. Проверь, что задачи остались на странице.

Задачи сохраняются в `localStorage` браузера.

## Про ошибку node --check

В этом окружении команда `node --check app/script.js` может завершаться ошибкой `Access is denied`.

Это не мешает приложению работать, потому что трекер запускается прямо в браузере через `app/index.html`. Для проверки достаточно открыть страницу и пройти шаги выше.
